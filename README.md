# Pokémon-katalog

## Projektbeskrivning
En Pokémon-katalog byggd med HTML, CSS och JavaScript. På startsidan visas en grid-layout med Pokémon-kort, där varje kort representerar en Pokémon. Katalogen har funktionerna sökning, filtrering och sortering. När man klickar på ett kort hamnar man på detaljsidan där man kan läsa mer om den specifika Pokémon. Det finns även en Team Builder-sida där man kan lägga till Pokémon och skapa ett eget team.

## Instruktioner för att köra projektet
Klona repot och öppna `index.html` i en webbläsare. Alternativt via VS Code med Live Server.
