# Pokédex & Team Builder

🔗 [Live Demo](your-github-pages-or-vercel-url)

## About
A Pokémon catalogue built with vanilla HTML, CSS and JavaScript during a 
JavaScript 1 course, focused on dynamic websites and API integration.

Browse a searchable, filterable Pokédex pulling live data from the PokéAPI,
then head to the Team Builder to assemble your squad and analyse its strengths,
weaknesses, type coverage and stat balance.

## My contributions
I owned the Team Builder page including UI, logic and accessibility. Features include
team and waitlist management with localStorage persistence, team analysis
(strengths, weaknesses, type coverage, stat balance) and Pokémon suggestions
based on your team's gaps.

## Running the project
Clone the repo and open `index.html` in a browser, or use VS Code Live Server.
